# 图标文件

这个文件夹需要包含以下PNG格式的图标文件：

- `icon16.png` - 16x16像素
- `icon48.png` - 48x48像素  
- `icon128.png` - 128x128像素

目前有一个SVG格式的图标文件 `icon.svg`，你可以使用在线工具将其转换为PNG格式：

1. 访问 https://convertio.co/svg-png/ 或其他SVG转PNG工具
2. 上传 `icon.svg` 文件
3. 分别转换为16x16、48x48、128x128像素的PNG文件
4. 将转换后的文件重命名为对应的文件名

或者你也可以使用任何图片编辑软件创建自己的图标。
